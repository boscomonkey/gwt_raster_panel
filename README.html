GWT Raster

This widget is a raster panel for drawing low-res pictures on a web page with the mouse. Built with GWT, this panel was originally designed to demonstrate how a signature widget can be implemented.

Usage is similar to a pencil or brush tool in a paint program. Move your mouse over the panel. When the left mouse button is down, the "dot" under the mouse cursor will turn on. By dragging the mouse without releasing the button, users can draw a trail of dots. When the left mouse button is released, no more dots will turn on.

The "Clear" button will turn off all the pixels in the panel.

Try it out now.

API

Create a RasterPanel with the width & height of the panel, the size of each "dot" (number of screen pixels), and whether you want a border. Add it to your DOM somewhere.

// 150x30 panel, each "dot" is 4 screen pixels high & wide, plus border
RasterPanel rasterPanel = new RasterPanel(150, 30, 4, true);
RootPanel.get("div1").add(rasterPanel);

Create a button to call RasterPanel#clear() upon mouse click and add that to the DOM.

Button btnClear = new Button("Clear");
btnClear.addClickListener(new ClickListener() {
    public void onClick(Widget sender) {
        rasterPanel.clear();
    }
});
RootPanel.get("div2").add(btnClear);

Call RasterPanel#getEngaged() to retrieve all the "dots" that are turned on.

HashSet engagedDots = rasterPanel.getEngaged()
